# kwonbk0223.github.io

Personal academic homepage of **Byeongkeun Kwon** — Ph.D. student at AAI Lab, Graduate School of Data Science, Kyungpook National University.

Built with Jekyll, hosted on GitHub Pages: <https://kwonbk0223.github.io>

## Structure

- `index.md` — Home (about, education, research interests, contact)
- `publications.md` — Journal papers, working papers, patents
- `projects.md` — Research projects
- `conferences.md` — Conference presentations
- `awards.md` — Awards, grants, media coverage
- `_layouts/default.html` — Layout & styles
- `assets/CV_ByeongkeunKwon.pdf` — CV

## How to update

Edit the `.md` files directly on GitHub (pencil icon) and commit — the site rebuilds automatically in ~1 minute.
